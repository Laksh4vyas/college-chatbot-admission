# 🚀 College Admission AI (Antigravity Project)

## 🧠 Overview

This is a **full-stack AI-powered web application** that helps users with college admission queries using a **Machine Learning ensemble model + chat interface**.

The system predicts responses based on user queries and provides guidance related to:

- Admissions 🎓  
- Cutoffs 📊  
- College suggestions 🏫  
- Deadlines ⏳  

---

## ✨ Features

- 💬 Interactive Chat UI (like ChatGPT)
- 🤖 ML Ensemble Model (Soft Voting: RF + LR + NB)
- 📊 Confidence Score Display
- ⚡ FastAPI Backend
- 🎨 Modern Frontend (Vite + React)
- 🐳 Fully Dockerized (Frontend + Backend)
- 🔄 Real-time API communication
